# Домашнее задание №4 ROS LiDAR

## Задание 2 - Базовое обнаружение препятствий

Запуск **roscore**

```
roscore
```

Запуск **лидара** 

```
rosrun ros_lidar_examples scan_publisher.py 
```

Запуск **ноды**

```
rosrun ros_lidar_examples task2.py
```
